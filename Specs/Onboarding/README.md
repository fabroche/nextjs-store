# Onboarding - Next.js Shopify Store

Bienvenido al proyecto **nextjs-store**. Esta documentación te ayudará a empezar rápidamente y entender los conceptos clave del proyecto.

## Orden de Lectura Recomendado

Si eres nuevo en el proyecto, te recomendamos leer la documentación en este orden:

### 1. [Getting Started](./01-getting-started.md)
Configuración inicial del proyecto, instalación de dependencias y primeros pasos.

**Tiempo estimado:** 15 minutos
**Temas:** Setup, variables de entorno, comandos básicos

### 2. [Project Overview](./02-project-overview.md)
Visión general del stack tecnológico y arquitectura del proyecto.

**Tiempo estimado:** 20 minutos
**Temas:** Stack tecnológico, estructura de carpetas, arquitectura

### 3. [Development Workflow](./03-development-workflow.md)
Guías prácticas para el desarrollo diario: cómo crear componentes, páginas y API routes.

**Tiempo estimado:** 30 minutos
**Temas:** Crear componentes, páginas, API routes, flujo de trabajo

### 4. [Coding Standards](./04-coding-standards.md)
Convenciones y estándares de código que se siguen en el proyecto.

**Tiempo estimado:** 25 minutos
**Temas:** TypeScript, SASS/BEM, naming conventions, best practices

### 5. [Shopify Integration](./05-shopify-integration.md)
Cómo trabajar con la integración de Shopify, servicios disponibles y endpoints.

**Tiempo estimado:** 20 minutos
**Temas:** Configuración de Shopify, servicios, API routes

### 6. [Troubleshooting](./06-troubleshooting.md)
Solución a errores comunes y problemas frecuentes.

**Tiempo estimado:** 10 minutos (referencia)
**Temas:** Errores comunes, soluciones, debugging
